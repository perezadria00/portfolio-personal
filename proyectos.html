Proyectos destacados

Durante el primer curso del Grado Superior de Desarrollo de Aplicaciones Web, realicé junto a otros compañeros un proyecto sobre productos ergonómicos. Este proyecto me permitió aplicar mis conocimientos técnicos y trabajar en equipo para desarrollar una solución completa. Incluyó:

  • Frontend: Diseño e implementación de la interfaz utilizando HTML, CSS y librerías como jQuery.
  • Backend: Desarrollo de funcionalidades dinámicas utilizando PHP.
  • CRUD y BBDD: Gestión de datos mediante MySQL para operaciones como crear, leer, actualizar y eliminar información de productos.

Este proyecto no solo mejoró mis habilidades técnicas, sino que también fortaleció mis capacidades de comunicación y colaboración, al trabajar de manera efectiva con mis compañeros para cumplir con los objetivos establecidos.
