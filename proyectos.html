Mis proyectos

A lo largo de mi formación, he trabajado en diversos proyectos que reflejan mi creatividad y habilidades técnicas. Algunos ejemplos destacados incluyen:

  • Gestión de Tareas: Una aplicación web que ayuda a los usuarios a organizar su tiempo con una interfaz intuitiva.
  • Web Responsive: Creación de sitios web adaptables para negocios locales con diseños modernos y optimización SEO.
  • Integración de APIs: Uso de APIs para crear herramientas con datos dinámicos, como información meteorológica y mapas interactivos.
